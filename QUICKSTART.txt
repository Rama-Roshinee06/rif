‚ö†Ô∏è TROUBLESHOOTING
==============================================================================

PROBLEM: "No PDF files found"
  ‚Üí Check: PDFs in input_pdfs/ folder
  ‚Üí Check: Filename ends with .pdf (not .PDF)

PROBLEM: "Excel file is empty"
  ‚Üí Check: extraction_status column - should be "SUCCESS"
  ‚Üí Try: python demo.py (test extraction)
  ‚Üí Try: python main.py --fallback (use simpler method)

PROBLEM: "Process is very slow"
  ‚Üí Normal: First run loads models (~1-2 min)
  ‚Üí Solution: Use fallback: python main.py --fallback
  ‚Üí Or: Install GPU support if you have NVIDIA GPU

PROBLEM: Module import errors
  ‚Üí Solution: Install missing package
    pip install -r requirements.txt

PROBLEM: Special characters in output (like ·ùé or (cid:172))
  ‚Üí Cause: PDF encoding issues (not your code)
  ‚Üí Solution: Try original PDF in Adobe Reader first
  ‚Üí Note: Pipeline extracts text as-is from PDF
